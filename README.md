# WordPressテーマ「Arkhē」

WEBサイト制作のベースに利用しやすく考慮したシンプルなWordPressテーマです。

GutenbergのコアブロックにCSSを加えるだけで、柔軟にコンテンツを組み立てることができるでしょう。

細かく面倒な部分はテーマに任せ、重要なことだけに集中してください。

公式サイト：https://arkhe-theme.com/

### テーマのアップデート方法

## Requirements
- WordPress 5.0 +
- PHP 5.6 +


## Browser support
- Chrome
- Safari
- Edge
- Firefox

Note: IE is not supported.


## 開発コマンド

パッケージインストール

```
npm ci
composer install
```


phpcs のコード規約セット

```
composer set-wpcs
```


ウォッチ & ビルド

```
npx gulp
```
